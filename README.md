# USS_AV_202605_FIIO0009_4007_S8TS

Aplicación Angular que consume una API pública y muestra publicaciones con comunicación entre componentes.

## Tecnologías
Angular 17 (aprox), Node.js 20+ (aprox)

## Ejecución
cd angular-s8-app  
npm install  
ng serve  
Abrir en http://localhost:4200

## Estructura
AppModule + FeaturesModule; servicios en core y componentes en features.
